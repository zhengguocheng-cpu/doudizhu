### 2. 简化我的排名指示器 ✅

#### 优化前
```css
.leaderboard-item.my-rank::before {
    content: '👤 我的排名';  /* 文字标签 */
    position: absolute;
    top: -14px;
    left: 20px;
    background: linear-gradient(...);
    padding: 6px 14px;
    font-size: 14px;
    /* ... */
}
```
**问题**:
- 有"👤 我的排名"文字标签
- 太显眼，视觉干扰
- 不够简洁

#### 优化后
```css
/* 完全移除 ::before 伪元素 */
/* 只保留 .my-rank 的高亮效果 */

.leaderboard-item.my-rank {
    background: linear-gradient(...);  /* 渐变背景 ✅ */
    border: 2px solid #667eea;         /* 紫色边框 ✅ */
    box-shadow: 0 4px 15px ...;        /* 阴影效果 ✅ */
}
```
**效果**:
- ✅ 移除文字标签
- ✅ 保留高亮背景
- ✅ 保留紫色边框
- ✅ 保留阴影效果
- ✅ 简洁清晰
