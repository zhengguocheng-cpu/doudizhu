# 游戏资源说明

## 📁 资源目录结构

```
resources/
├── images/                 # 图片资源
│   ├── ui/                # UI图片
│   │   ├── buttons/       # 按钮图片
│   │   ├── panels/        # 面板背景
│   │   ├── icons/         # 图标
│   │   └── backgrounds/   # 背景图片
│   ├── cards/             # 扑克牌图片
│   │   ├── spades/        # 黑桃
│   │   ├── hearts/        # 红桃
│   │   ├── diamonds/      # 方块
│   │   ├── clubs/         # 梅花
│   │   └── jokers/        # 大小王
│   ├── avatars/           # 头像图片
│   └── effects/           # 特效图片
├── audio/                 # 音频资源
│   ├── music/             # 背景音乐
│   ├── sounds/            # 音效
│   └── voices/            # 语音
├── animations/            # 动画资源
│   ├── cards/             # 卡片动画
│   ├── effects/           # 特效动画
│   └── ui/                # UI动画
└── prefabs/               # 预制体
    ├── ui/                # UI预制体
    ├── cards/             # 卡片预制体
    └── effects/           # 特效预制体
```

## 🎨 图片资源规格

### UI图片
- **按钮图片**: 128x128px, PNG格式
- **面板背景**: 1024x1024px, PNG格式
- **图标**: 64x64px, PNG格式
- **背景图片**: 1920x1080px, JPG格式

### 扑克牌图片
- **卡片尺寸**: 120x168px, PNG格式
- **卡片背面**: 120x168px, PNG格式
- **卡片正面**: 按花色分类存储

### 头像图片
- **头像尺寸**: 128x128px, PNG格式
- **圆形头像**: 带透明背景
- **方形头像**: 带圆角效果

## 🎵 音频资源规格

### 背景音乐
- **格式**: MP3或OGG
- **时长**: 2-5分钟循环
- **音量**: 适中，不干扰游戏

### 音效
- **格式**: WAV或MP3
- **时长**: 0.5-3秒
- **音量**: 清晰可辨

### 音效列表
- `card_click.wav` - 卡片点击音效
- `card_play.wav` - 出牌音效
- `win.wav` - 胜利音效
- `lose.wav` - 失败音效
- `button_click.wav` - 按钮点击音效
- `notification.wav` - 通知音效
- `game_start.wav` - 游戏开始音效
- `game_end.wav` - 游戏结束音效

## 🎯 资源使用指南

### 图片资源
1. 所有图片资源应放在对应的子目录下
2. 图片命名应遵循规范，便于查找
3. 建议使用WebP格式以减小文件大小
4. 高分辨率设备需要提供@2x和@3x版本

### 音频资源
1. 背景音乐应循环播放
2. 音效应短小精悍，避免过长
3. 音量应适中，不干扰游戏体验
4. 支持多种格式以确保兼容性

### 动画资源
1. 动画应流畅自然
2. 时长应适中，避免过长
3. 应支持循环播放
4. 性能优化，避免卡顿

## 🔧 资源优化建议

### 图片优化
- 使用适当的压缩比
- 移除不必要的透明通道
- 合并小图片为图集
- 使用九宫格拉伸

### 音频优化
- 选择合适的采样率
- 使用适当的比特率
- 预加载常用音效
- 及时释放不用的音频

### 动画优化
- 使用骨骼动画减少顶点数
- 避免过于复杂的粒子效果
- 合理使用缓动函数
- 及时清理动画对象

## 📝 资源命名规范

### 图片命名
- UI图片: `ui_功能_状态.png`
- 卡片图片: `card_花色_点数.png`
- 头像图片: `avatar_编号.png`
- 特效图片: `effect_类型_编号.png`

### 音频命名
- 背景音乐: `bgm_场景名.mp3`
- 音效: `sfx_功能名.wav`
- 语音: `voice_角色_内容.wav`

### 动画命名
- 卡片动画: `anim_card_动作名`
- 特效动画: `anim_effect_类型名`
- UI动画: `anim_ui_组件名`

## 🚀 资源加载策略

### 预加载
- 关键资源在游戏启动时预加载
- 场景相关资源在场景加载时预加载
- 用户操作相关资源在需要时加载

### 懒加载
- 非关键资源延迟加载
- 根据用户行为动态加载
- 及时释放不用的资源

### 缓存策略
- 常用资源常驻内存
- 临时资源及时释放
- 根据内存使用情况调整缓存策略
