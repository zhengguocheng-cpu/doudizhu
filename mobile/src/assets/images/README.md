# 图片资源说明

## 目录结构
```
assets/images/
├── logo.png              # 应用Logo
├── icons/                # 图标资源
│   ├── lobby.png         # 大厅图标
│   ├── lobby-active.png  # 大厅激活图标
│   ├── game.png          # 游戏图标
│   ├── game-active.png   # 游戏激活图标
│   └── avatar/           # 头像资源
│       ├── avatar1.png
│       ├── avatar2.png
│       └── ...
└── cards/                # 扑克牌图片
    ├── spades/           # 黑桃
    ├── hearts/           # 红桃
    ├── diamonds/          # 方块
    └── clubs/             # 梅花
```

## 图片规格要求

### Logo图片
- 尺寸: 120x120px
- 格式: PNG
- 背景: 透明
- 用途: 应用启动页、登录页

### 图标资源
- 尺寸: 48x48px (普通), 96x96px (高分辨率)
- 格式: PNG
- 背景: 透明
- 用途: TabBar图标

### 头像资源
- 尺寸: 64x64px
- 格式: PNG
- 背景: 透明或圆形
- 用途: 用户头像选择

### 扑克牌图片
- 尺寸: 60x84px
- 格式: PNG
- 背景: 白色
- 用途: 游戏中的扑克牌显示

## 使用说明

1. 所有图片资源应放在对应的目录下
2. 图片命名应遵循规范，便于维护
3. 建议使用WebP格式以减小文件大小
4. 高分辨率设备需要提供@2x和@3x版本
