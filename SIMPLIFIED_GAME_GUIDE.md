# 🎮 简化版斗地主游戏使用说明

## ✅ 认证系统已完全移除！

**现在游戏超级简单：**
1. 输入用户名
2. 点击"进入游戏大厅"
3. 选择房间进入游戏
4. 开始玩！

## 🚀 快速开始

### **1. 启动游戏**
```bash
# 启动后端服务器
cd backend
npx ts-node --transpile-only --project ts-node.json server.ts

# 打开前端页面
# http://localhost:3000/login/index.html
```

### **2. 进入游戏**
1. 打开浏览器访问：`http://localhost:3000/login/index.html`
2. 输入任意用户名（比如：`玩家1`）
3. 点击"进入游戏大厅"
4. 选择任意房间进入
5. 开始游戏！

## 🔧 技术说明

### **前端架构**
- **无认证**：直接传递用户名参数
- **单连接**：GlobalSocketManager管理全局连接
- **简化状态**：只保留游戏相关状态

### **后端架构**
- **无认证中间件**：移除所有认证检查
- **直接处理**：收到请求直接处理游戏逻辑
- **简化权限**：所有用户都有所有权限
